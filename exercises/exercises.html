Hora de Praticar

A prática é a chave para o domínio de qualquer habilidade. Por isso, chegou a hora de colocar seus conhecimentos em teste.

1. Converta os números decimais para o sistema octal:

  1. 25
  2. 63
  3. 128
  4. 512

2. Converta os números do sistema octal para o decimal:

  1. 25
  2. 123
  3. 452
  4. 458

3. Qual é a representação octal do número decimal 777?

  1. 1371
  2. 1457
  3. 1411
  4. 1417

4. Converta o número octal 152 para o sistema decimal.

  1. 215
  2. 230
  3. 237
  4. 240

5. Qual é o valor decimal do número octal 547?

  1. 355
  2. 360
  3. 359
  4. 380

6. No número octal 643 qual é o dígito mais significativo significativo?

  1. 6
  2. 4
  3. 3
  4. Não há
